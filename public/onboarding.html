Welcome to Quick Query

Quickly replace Query Parameters on your most used websites.

https://example.com?q={KEY} becomes https://example.com?q=mySearchTerm

Getting Started

  1. Add URLs

    • Click the extension icon in the top-right corner of your browser to manage your URLs.

    • Add, modify, or delete URL templates. Each URL template is identified by a unique keyword to identify it.

    • Optionally Pin the Extension for quick access.

  2. Use Quick Query

    • Type q followed by a keyword and your search term in the address bar. Notice quotation marks allow multiple words to act as either a keyword or search term.
      • q myKeyword mySearchTerm
      • q "my Keyword" "my search term"
      • q myKeyword "my search term"
      • q myKeyword my search term
    • If no keyword is entered, the default template is used.
      • q mySearchTerm
      • q my search term

  3. View Results

    After entering q myKeyword mySearchTerm in your URL Search Box, mySearchTerm will replace {KEY} in your URL Template corresponding to your Keyword.

    Example:

    • https://example.com?q={KEY} becomes https://example.com?q=mySearchTerm

How Quick Query Can Be Useful

Here are a handful of untested samples of to give you ideas of how you may want to leverage this extension. Please look at a live URL before creating your URL Template.

  • Quick Access to Jira Issues:

    https://yourcompany.atlassian.net/browse/{KEY}.

    Use Case: Type q atlas PROJ-123 in the address bar to directly access Jira issue PROJ-123.

  • Search for GitHub Repositories:

    https://github.com/search?q={KEY}.

    Use Case: Enter q "github search" react components to search for repositories related to React components on GitHub.

  • Find Bitbucket Repositories:

    https://bitbucket.org/repo/all?name={KEY}.

    Use Case: Type q bb frontend to search for repositories related to frontend development on Bitbucket.

  • Look Up Contacts in Salesforce:

    https://login.salesforce.com/001/o?search={KEY}.

    Use Case: Enter q "sf contact" John Doe to quickly search for the contact "John Doe" in Salesforce.

  • Find Opportunities in HubSpot CRM:

    https://app.hubspot.com/contacts/{KEY}.

    Use Case: Type q hub 12345 to access the details of opportunity 12345 in HubSpot CRM.

  • Access Tickets in Zendesk:

    https://yourcompany.zendesk.com/agent/tickets/{KEY}.

    Use Case: Enter q zen 56789 to view Zendesk ticket #56789.

  • Search Issues in GitLab:

    https://gitlab.com/search?search={KEY}.

    Use Case: Type q bug report to search for issues related to bug reports on GitLab.

  • Navigate to Asana Tasks:

    https://app.asana.com/0/search/{KEY}.

    Use Case: Enter q asana planning to find tasks related to project planning in Asana.
